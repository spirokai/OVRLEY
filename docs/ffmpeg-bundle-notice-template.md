# FFmpeg Bundle Notice Template

Portable release archives generated by `scripts/package-portable.mjs` include a generated `THIRD_PARTY_NOTICES.txt`.

That generated notice applies to release artifacts that actually contain a downloaded FFmpeg binary under `vendor/ffmpeg`. The source repository does not commit or vendor FFmpeg binaries.

The generated notice records:

- that FFmpeg is included as an unmodified separate command-line component
- where FFmpeg source and license information can be found
- the FFmpeg binary provider used by the install script
- the actual bundled `ffmpeg -version` output
- the actual bundled `ffmpeg -L` output
