# Theme showcase design plan

**Goal:** To create a set of theme suggestions for the app that can be used as a showcase for website. The themes should cover a range of styles and color schemes, and should be designed to work well with the app's existing UI elements. Total of exactly 16 schemes is required.

**We must cover scenarios that include these combinations of route, elevation, and metric widgets:**

- with/without route/elevation
- with/without elevation area
- different colors of remaining/completed distance
- different colors/opacity of elevation area
- different shadows, most designs without shadows, some with on
- with/without gradient widget
- text borders, most designs without borders, some with on
- with/without icons
- units with different color
- 1 scheme where fonts and labels use different fonts
