# Theme showcase design plan

**Goal:** To create a set of theme suggestions for the app that can be used as a showcase for website. The themes should cover a range of styles and color schemes, and should be designed to work well with the app's existing UI elements. Total of exactly 16 schemes is required.

**We must cover scenarios that include these combinations of route, elevation, and metric widgets:**

- with/without route/elevation
- with/without elevation area
- different colors of remaining/completed distance
- different colors/opacity of elevation area
- different shadows, most designs without shadows, some with on
- with/without gradient widget
- text borders, most designs without borders, some with on
- with/without icons
- units with different color
- 1 scheme where fonts and labels use different fonts

## Color Combinations and styles

**We need exactly 10 standard color schemes that cover a range of widgets, styles using these three neutral color combinations:**

- SAFA Brian theme - pure white (Evogria font required)
- neutral theme - pure white (any font except Evogria)
- Champagne (#FFF2C9) + rust/clay (#C54A2B; accents)

**We need exactly one scheme for each of the following unique combinations (total 6):**

- Cyan (#00E5FF) + light blue (#7DF9FF) futuristic hud
- Burnt orange (#FF6A2B) + cream orange (#DCB9A6; just minor accents) - DONE; screen8
- Neon acid lime (#D7FF3F) + titanium gray (#C6CDD5) - DONE; screen12
- Blue navy (#95C4EC) + neon teal (#11E9C5)
- Lavender (#CAB2FB) + slate brown (#8B5CF6)

## Non-negotiable settings and fonts

**Non-negotiable scene settings:**

- resolution 3840x2160px
- global scale 1
- maximum shadows/distance - 4
- maximum border width - 2

**Permissable fonts - any designs using different fonts will be rejected:**

- Bebas Neue
- Evogria
- Furore
- Saira Stencil
- Teko
- Inter black
